PegasusEngine

Custom 3D game engine built on C# and OpenGL for experimenting with graphics and physics.

C# (.NET) OpenGL OpenTK ImGui Game Engine 2024 - Today
Visit GitHub

About the project

PegasusEngine started as a passion project aimed at understanding what happens "under the hood" of modern game engines like Unity or Unreal. Instead of using out-of-the-box solutions, I decided to implement the core systems from scratch using C# and the OpenTK library (a wrapper for OpenGL).

The current version of the engine supports basic 3D rendering, model loading, a camera system, and simple scene management. In the future, I plan to add a physics engine and a scene editor with a graphical user interface.

Key features

  • Custom Rendering Pipeline: Implementation of shaders, buffers and rendering right in OpenGL.
  • Scene Management: Hierarchy of game objects (Parent-Child relations).
  • Component System: Architecture inspired by ECS (Entity Component System) for flexibility with game objects.
  • Input System: Abstrakcia vstupu z klávesnice a myši.

Tech Stack

  • Language: C# (.NET 8)
  • Graphics: OpenGL 4.x
  • Wrapper: OpenTK
  • UI: ImGui.NET
  • Math: OpenTK.Mathematics
