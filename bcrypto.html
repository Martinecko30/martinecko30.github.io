bCrypto

Kryptomenový plugin pre minecraftový server s rozšírením ekonomiky.

Java Paper MC SQLite 2024 - Dnes
Pozrieť web Basicland

O projekte

bCrypto je plugin pre Minecraft server Basicland.cz. Plugin podrpouje ekonomiku a rozvíja ju o možnosti kryptomien ktoré sú napojené na reálny svet. Umožnuje hráčov vytvoriť si vlastnú peňaženku, nakupovať a predávať kryptomeny a pozerať si históriu zmien.

Kľúčové vlastnosti

  • Kryptomeny: Plugin je napojený na API a všetky ceny sú v reálnom čase.
  • Peňaženky: Hráč si vytvára vlastnú peňaženku z vlastnou ekonomikou prepojenou na server.

Tech Stack

  • Jazyk:Java (JDK 21)
  • Engine:Paper MC
  • Databáza: SQLite
  • Build Tool: Gradle
  • Automation Tool: Lombok
